## 风险与注意事项

| 风险 | 表现 | 处理 |
|---|---|---|
| 行为 log 文件名不匹配 | r04 报错或截断 | 检查 `data/exp1_sess1_log/` 和 `data/exp1_sess2_log/` 中对应被试的 .txt 文件是否存在 |
| 某被试 trial 数异常少 | r02/r03 输出中该被试比例异常 | 记录但不阻塞；可能是该被试数据质量问题 |
| cluster perm 结果与论文不完全一致 | p 值有差异 | 正常——论文可能用了不同参数或全部 trial；记录实际 p 值 |
| MATLAB 运行时间过长 | 超过 25 分钟 | 检查是否卡在某个被试；必要时用 `run_all('from','r04')` 跳过已完成步骤 |
| 内存不足 | MATLAB 报 out of memory | 23 个被试的 .mat 文件总量约 1-2 GB，一般不会触发；如触发可分批运行 |
